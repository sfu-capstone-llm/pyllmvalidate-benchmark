@property
# TODO: X
@property

    pass

@@ -1,6 +1,8 @@
 @property
 # TODO: X
+
+
 @property
 def foo():
     pass
Where flake8 then complains:

stdin:5:1: E304 blank lines found after function decorator

Operating system: Arch Linux
Python version: 3.6.5
Black version: 18.4a2
Does also happen on master: Haven't checked.
